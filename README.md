# 18. REST API — автотесты для книжного клуба

## Как устроен проект

- **`.env`** — настройки (URL, таймауты), в репозиторий не попадает
- **`config.py`** — читает `.env` и отдаёт настройки в тесты
- **`pytest.ini`** — метки для тестов (smoke, regression)
- **`schemas/club_list_schema.py`** — JSON-схема: описывает, как должен выглядеть ответ API
- **`tests/conftest.py`** — фикстура `base_url` (отдаёт URL в тесты) и хук для Allure
- **`tests/test_clubs_list.py`** — сами тесты:
  - получение списка клубов
  - проверка схемы
  - пагинация (размер страницы и переход по страницам)
  - поиск по названию
  - проверка обязательных полей
